HeySupport Widget Demo

The chat widget is loaded in the bottom-right corner. Click the bubble to try it out.

Embed code:

<script src="https://cdn.jsdelivr.net/gh/heysmmprovider/heysupport@latest/dist/heysupport.min.js" data-server="https://your-socket-server.com"></script>
Note: This demo connects to http://localhost:3000 by default. Make sure your HeySupport socket server is running. See heysupport-socket for setup instructions.
